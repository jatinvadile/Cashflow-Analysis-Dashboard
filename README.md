# Cashflow-Analysis-Dashboard
Automated cashflow tracking and analysis across multiple bank statements and currency datasets (TZS & USD) to improve financial visibility and support faster decision-making.

▸ Built a unified data pipeline using SQL and Power BI, consolidating data from multiple bank statements and sources into a single reporting layer.
▸ Performed end-to-end data cleaning and transformation - resolved inconsistencies across TZS and USD currency datasets, improving data accuracy to 98%+.
▸ Designed interactive Power BI dashboards visualizing cash inflow/outflow trends, balance movements, and currency-wise performance for stakeholders.
▸ Reduced manual reporting effort by 40% and enabled near real-time cash monitoring - accelerating financial decision-making for management.
▸ Automated data refresh and transformation using Python (Pandas) and Power Query, eliminating repetitive manual steps in the reporting workflow.
